# CRYPTO-DASHBOARD
Cryptocurrency Market Analysis Dashboard
This project presents an interactive Cryptocurrency Market Analysis Dashboard built using Microsoft Power BI.
The dashboard analyzes historical cryptocurrency data and provides insights into price movements, trading volume, and market capitalization trends.
The project uses Power Query for data transformation, KPI indicators for key metrics, forecasting for trend prediction, and interactive visualizations for data exploration.
The dashboard analyzes multiple cryptocurrencies including Bitcoin, Ethereum, Dogecoin, Cosmos, and Binance Coin.

**🎯 Project Objective**
The objective of this project is to analyze cryptocurrency market trends and present them in an interactive dashboard to help understand:
Price fluctuations over time
Market capitalization trends
Trading volume patterns
Daily price volatility

**🛠 Tools & Technologies Used**
Microsoft Power BI
Power Query (Data Cleaning & Transformation)
KPI Indicators
Card Visuals
Forecasting Analysis
Interactive Data Visualization

**⚙ Data Preparation**
Data preparation was performed using Power Query where the dataset was cleaned and transformed.
The following steps were performed:
Handling missing values
Formatting date fields
Structuring financial metrics
Preparing the dataset for visualization

**📈 Dashboard Features**
Price Trend Analysis
The dashboard includes Open vs Close price charts to analyze daily price movement and identify trends in the cryptocurrency market.
High–Low Price Analysis
The High and Low chart helps measure the volatility of the market and shows how much the price fluctuates within a specific period.
Market Capitalization Analysis
The Marketcap by Date visualization shows how the overall value of cryptocurrencies changes over time.

This helps identify:
Market growth phases
Market corrections
Bull and bear cycles
Trading Volume Analysis
The Daily Volume chart shows trading activity across time.
Higher trading volume indicates stronger market participation.
Forecasting Analysis
Power BI forecasting feature is used to estimate future market trends based on historical price data.
This helps in understanding potential future price movement patterns.

**📊 Key Metrics (KPI Indicators)**
The dashboard includes KPI cards showing important metrics:
Highest Price
Lowest Price
Average Price
Total Trading Volume
Total Market Capitalization
These indicators provide a quick overview of the cryptocurrency market performance.

**🎛 Interactive Dashboard Filters**
The dashboard includes interactive filters allowing users to explore the data dynamically.
Cryptocurrency Filter
Users can switch between cryptocurrencies such as:
Bitcoin
Ethereum
Dogecoin
Cosmos
Binance Coin
Year Filter
Users can analyze market trends from 2013 to 2021.
